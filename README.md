# David Liechty — Personal Homepage

A minimal, responsive personal homepage built with plain HTML and CSS. No frameworks, no build step.

## Features

- Single-file site (`index.html`) with inlined CSS
- Automatic light/dark mode via `prefers-color-scheme`
- Responsive layout for mobile and desktop
- Sections: hero, projects, about, and contact

## Development

Open `index.html` in a browser — no build tools or dependencies required.

## Deployment

Hosted with [GitHub Pages](https://pages.github.com/). Push to `main` to deploy.
